# FIFA World Cup 2026 Bracket Template

Interactive FIFA World Cup 2026 bracket and fixtures template built with HTML, CSS, and JavaScript.

## Features
- Editable group standings
- Interactive fixtures
- Timezone converter
- Knockout bracket
- LocalStorage autosave
- Printable / Export PDF layout

## Project Structure

```bash
world-cup-2026-project/
│
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   └── js/
│       └── app.js
```

## Usage

Open `index.html` in your browser.

## Deployment

Compatible with:
- GitHub Pages
- Vercel
- Netlify

## Security Notes

- No backend code
- No external API calls
- Uses browser localStorage only
